# Site do Centro Esportivo e Cultural Luís Antônio
Este projeto teve como objetivo apresentar e divulgar informações sobre o projeto social do CECLA.
Além do desenvolvimento, todas as etapas do projeto forma feitas por mim.


Tecnologias utilizadas:

- **HTML5** — Estrutura semântica e otimizada para SEO  
- **CSS3** — Design responsivo, animações e layout adaptável  
- **JavaScript** — Interatividade e efeitos dinâmicos  
- **Bootstrap** — Apenas para o uso de ícones responsivos
- **Hosting** — Hospedagem configurada pela plataforma da *Hostinger*

Destaques do projeto

- Desenvolvido com base no conceito Mobile First Design
- Layout moderno e leve  
- Sessões modulares pensando na fácil manutenção  
- Código limpo e bem estruturado  
- Hospedagem configurada por mim com domínio personalizado
